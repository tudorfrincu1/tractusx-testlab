# Tractus-X TestLab

Visual test authoring tool for Eclipse Tractus-X dataspaces.

## What is TestLab?

TestLab lets you build, validate, and run dataspace integration tests visually — no code required. Drag blocks, connect them, and generate executable YAML test definitions.
