# Tractus-X TestLab

Test authoring and execution library for Eclipse Tractus-X dataspaces.

## What is TestLab?

TestLab lets you author, validate, and run dataspace integration tests. You write declarative YAML test definitions and TestLab compiles, validates, and executes them against real or mocked connectors.
