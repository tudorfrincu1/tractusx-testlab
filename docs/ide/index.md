# IDE

Getting started with the TestLab visual block editor.

## Launch

```bash
cd ide && npm install && npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Layout

The IDE has two panels:

- **Left (60%)** — Blockly workspace where you drag and connect blocks
- **Right (40%)** — Live YAML preview (read-only) showing the generated test definition
